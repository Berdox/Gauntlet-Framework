# Gauntlet Framework

A framework for Titanfall 2 that provides functions for creating gauntlets.  
It handles triggers, UI, logic, and data management so custom gauntlets can be made easier.

## Features

- **Start / Finish Logic**  
  Automatic setup and event handling for beginning and ending a run.

- **Checkpoint System**  
  WIP

- **Timer**  
  Built-in HUD elements for displaying current and final time.

- **UI Management**  
  WIP

- **Leaderboard System**  
 WIP

- **Configurable Setup**  
 WIP

## Purpose

This framework provides a consistent structure for custom Titanfall 2 gauntlets. For parkour routes, surf maps, or full gauntlet map.
